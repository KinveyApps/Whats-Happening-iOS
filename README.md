Whats-Happening-iOS
===================

Sample App Showing Today Widget Extensions with KinveyKit

## How to use
1. Build and run
2. Swipe down notification center
3. Scroll to the bottom, tap "Edit" and add WhatsHappening

## Requirements

* Requires XCode 6+, iOS 8+
